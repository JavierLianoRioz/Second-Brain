# Hipótesis Nula y Alternativa

En todo contraste se plantean dos hipótesis enfrentadas.

## 1. Hipótesis Nula ($H_0$)
Es la hipótesis que se desea contrastar y que mantendremos como cierta a menos que los datos indiquen lo contrario.
- **Presunción de inocencia**: Se asume cierta por defecto.
- **Operadores**: Siempre contiene el signo de igualdad ($=$, $\leq$ o $\geq$).
- **Decisión**: Nunca se "acepta", solo se "rechaza" o "no se rechaza".

## 2. Hipótesis Alternativa ($H_1$)
Es la negación de $H_0$ y generalmente lo que el investigador intenta demostrar.
- **Operadores**: Nunca contiene el signo de igualdad ($<$, $>$ o $\neq$).
- **Decisión**: Se acepta o no se acepta.
