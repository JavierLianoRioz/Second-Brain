# p-value (Nivel Crítico)

El **p-value** (o nivel crítico $p$) es el nivel de significación más pequeño para el que la muestra obtenida obligaría a rechazar $H_0$.

## Interpretación y Toma de Decisiones
Se compara el p-value con el nivel de significación $\alpha$ fijado:

- **Si $p \leq \alpha$**: El resultado es **significativo**. Hay evidencia suficiente para rechazar $H_0$. Se acepta $H_1$.
- **Si $p > \alpha$**: El resultado es **no significativo**. No se puede rechazar $H_0$.

> [!TIP]
> Cuanto más pequeño sea el p-value, mayor es la evidencia en contra de la hipótesis nula.
