# Tipos de Variables

Una **variable** es una característica observable que varía entre los individuos de una población. Las posibles formas que toma se llaman **modalidades**.

## 1. Variables Cualitativas
No se pueden asociar naturalmente a un número (no admiten operaciones algebraicas).
- **Nominales**: No admiten orden (ej. Grupo sanguíneo, religión).
- **Binarias (Dicotómicas)**: Solo dos posibilidades (ej. Sexo, fuma sí/no).
- **Ordinales**: Sus valores se pueden ordenar (ej. Grado de satisfacción, intensidad del dolor).

## 2. Variables Cuantitativas (Numéricas)
Los valores son números (admiten operaciones algebraicas).
- **Discretas**: Valores finitos o contables, usualmente números enteros (ej. Número de hijos).
- **Continuas**: Valores infinitos entre dos extremos, usualmente con decimales (ej. Altura, edad exacta).

## Requisitos de las Modalidades
Para que un sistema de categorías sea válido, debe ser:
1. **Exhaustivo**: No olvidar ningún valor posible.
2. **Excluyente**: Un individuo no puede presentar dos valores simultáneos.
