# Representaciones Gráficas

Visualización de datos según el tipo de variable.

## Para Variables Cualitativas
- **Diagrama de Barras**: Altura proporcional a la frecuencia.
- **Diagrama de Sectores (Tarta)**: Áreas proporcionales (ideal para 2-5 valores).
- **Gráfico de Mosaicos**: Relación entre varias variables sobre un mismo individuo.

## Para Variables Cuantitativas
- **Diagrama de Barras**: Para discretas, similar a cualitativas.
- **Histograma**: Para continuas. Barras juntas que inician y terminan en los límites de clase.
- **Diagrama de Línea**: Variación de una magnitud respecto a otra continua.
- **Gráfico de Series**: Diagrama de línea donde el eje X es el tiempo.
- **Gráfico de Dispersión**: Puntos $(x,y)$ para ver relación entre dos variables numéricas.
- **Gráfico de Burbuja**: Dispersión con 3ª y 4ª dimensión (tamaño y color).
- **Diagrama de Cajas (Box Plot)**: Especialmente útil para visualizar:
    - Valores mínimo y máximo.
    - Cuartiles $Q_1$ (0.25), $Q_2$ (Mediana, 0.50) y $Q_3$ (0.75).
    - Existencia de **valores atípicos** (outliers).
    - Simetría o asimetría de la distribución.
