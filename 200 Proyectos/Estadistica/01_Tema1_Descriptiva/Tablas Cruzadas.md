# Tablas Cruzadas (Tabulación Cruzada)

Una **tabulación cruzada** es un resumen tabular de la relación entre los datos de **dos o más variables**.

## Características
- Los datos deben estar agrupados en clases o categorías.
- Se puede emplear para:
    - Relacionar dos variables cualitativas.
    - Relacionar variables cuantitativas discretas.
    - Relacionar variables cuantitativas continuas (previa agrupación en clases).
- Permite visualizar cómo se distribuyen las frecuencias de una variable en función de otra (ej: Sexo vs Nivel Educativo).
