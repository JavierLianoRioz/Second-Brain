# Estimación Puntual

Consiste en obtener un único valor numérico para aproximar un parámetro poblacional.

## Conceptos Clave
- **Parámetro**: Valor de la población (ej: media $\mu$, proporción $P$, varianza $\sigma^2$).
- **Estimador**: Valor calculado en la muestra ($\bar{x}$, $p$, cuasivarianza $\hat{s}^2$).
- **Estimación**: El valor concreto que toma el estimador para una muestra dada.

## Características de un Buen Estimador
1. **Consistencia**: A mayor tamaño de muestra, más se acerca al valor real.
2. **Carencia de Sesgo**: El valor medio de las estimaciones coincide con el parámetro.
3. **Eficiencia**: Tiene la mínima varianza posible (mínima dispersión).
4. **Suficiencia**: Aprovecha toda la información de la muestra.
