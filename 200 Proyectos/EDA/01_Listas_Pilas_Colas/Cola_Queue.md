# Cola (Queue)

Una **Cola** es un ADT que sigue el principio **FIFO** (*First In, First Out*). El primero en entrar es el primero en salir.

## Complejidad Big O

| Operación | Complejidad | Nombre Común |
| :--- | :--- | :--- |
| **Inserción** | $O(1)$ | `enqueue()` |
| **Edición** | $O(n)$ | No es común. |
| **Eliminación** | $O(1)$ | `dequeue()` |
| **Peek** | $O(1)$ | `front()` |
