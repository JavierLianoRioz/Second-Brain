# Políticas de Seguridad y Gestión de Incidentes

Marco organizativo para mantener la seguridad a largo plazo.

## Políticas Comunes
- **Control de Accesos**: MFA (Multi-Factor Authentication), gestión de roles.
- **Uso Aceptable**: Reglas para empleados.
- **Backup y Recuperación**: Frecuencia y pruebas de restauración.

## Gestión de Incidentes (Ciclo NIST)
1. **Preparación**: Planificación y herramientas.
2. **Detección y Análisis**: Identificar que algo está pasando.
3. **Contención, Erradicación y Recuperación**: Detener el ataque, limpiar y volver a la normalidad.
4. **Post-incidente**: Lecciones aprendidas para mejorar el sistema.
