# Álgebra Relacional

El **Álgebra Relacional** es un lenguaje de consulta procedimental que consta de un conjunto de operaciones que toman una o dos relaciones como entrada y producen una nueva relación como resultado.

### Características
*   **Cerradura:** El resultado de una operación es siempre una relación.
*   **Base de SQL:** Aunque SQL es más amplio, su núcleo se basa en estas operaciones.

Se divide en **operadores fundamentales** (selección, proyección, unión, diferencia, producto cartesiano, renombramiento) y **operadores derivados** (intersección, join, división).
