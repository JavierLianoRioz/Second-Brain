# Álgebra Relacional

Es un lenguaje de consulta **procedural** que define una serie de operaciones sobre relaciones ([Modelo_Relacional_Conceptos](../02_Dise%C3%B1o/Modelo_Relacional_Conceptos.md)) para obtener una nueva relación como resultado.

Es la base teórica de [SQL_DML](../03_SQL/SQL_DML.md).


## Relación con SQL
El Álgebra Relacional proporciona la base teórica para el lenguaje [[SQL_SELECT]].
*   La **Selección** equivale al `WHERE` ([SQL_WHERE](../03_SQL/SQL_WHERE.md)).
*   La **Proyección** equivale al `SELECT` (lista de columnas).
*   La **Reunión (Join)** equivale al `JOIN` ([SQL_JOIN](../03_SQL/SQL_JOIN.md)).
