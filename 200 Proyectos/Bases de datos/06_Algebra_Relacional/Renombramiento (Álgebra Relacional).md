# Renombramiento (Álgebra Relacional)

El **Renombramiento** (denotado por **ρ**, rho) es una operación unaria que permite cambiar el nombre de una relación o de sus atributos.

Es útil para evitar ambigüedades (por ejemplo, al hacer Self-Joins o uniones de tablas con mismos nombres) o para clarificar el resultado.

### Sintaxis
`ρ NuevoNombre (R)` o `ρ NuevoNombre(A1, A2...) (R)`
