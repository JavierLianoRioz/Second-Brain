# División (Álgebra Relacional)

La **División** (denotada por **÷**) es una operación utilizada para consultas del tipo "para todos".
Devuelve las tuplas de una relación R que están asociadas con **todas** las tuplas de una relación S.

### Sintaxis
`R ÷ S`

### Ejemplo
Si tenemos:
*   **R (Alumno, Asignatura)**: Lista qué asignaturas cursa cada alumno.
*   **S (Asignatura)**: Lista todas las asignaturas obligatorias.

`R ÷ S` devolverá los alumnos que cursan **todas** las asignaturas obligatorias.
