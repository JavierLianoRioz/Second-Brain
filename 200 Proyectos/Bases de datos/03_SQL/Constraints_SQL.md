# Constraints SQL

Reglas que garantizan la calidad de los datos en [SQL DDL](SQL_DDL.md).

*   **PRIMARY KEY (PK)**: Identificador único. Ver [Clave Primaria](../02_Dise%C3%B1o/Clave_Primaria.md).
*   **FOREIGN KEY (FK)**: Enlace a otra tabla. Ver [Clave Foranea](../02_Dise%C3%B1o/Clave_Foranea.md).
*   **NOT NULL**: Obliga a tener un valor.
*   **UNIQUE**: Impide valores duplicados.
*   **CHECK**: Valida una condición lógica.
*   **DEFAULT**: Asigna un valor por defecto.
