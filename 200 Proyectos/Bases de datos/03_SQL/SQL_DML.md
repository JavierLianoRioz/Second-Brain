# SQL DML (Data Manipulation Language)

El **DML** permite gestionar los datos almacenados: insertar, modificar y borrar.

## Comandos Principales
*   [[INSERT]]: Añadir filas.
*   [[UPDATE]]: Modificar filas.
*   [[DELETE]]: Borrar filas.
*   [[SELECT_Basico]]: Consultar filas.
