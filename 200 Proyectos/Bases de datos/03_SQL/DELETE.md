# DELETE

Comando [SQL_DML](SQL_DML.md) para eliminar filas.

> [!CAUTION]
> Siempre usa [SQL_WHERE](SQL_WHERE.md). Si lo olvidas, borrarás **todos** los datos.

## Sintaxis
```sql
DELETE FROM tabla WHERE condicion;
```

## Ejemplo
```sql
DELETE FROM cliente WHERE id = 1;
```
