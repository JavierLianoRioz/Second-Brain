# DCL (Data Control Language)

El **DCL** es el [[Sublenguajes de SQL|sublenguaje de SQL]] utilizado para controlar los permisos de acceso a los datos en la base de datos.

### Comandos Principales

*   `GRANT`: Otorga permisos específicos (como `SELECT`, `INSERT`, `UPDATE`) a un usuario o rol sobre un objeto de la base de datos.
*   `REVOKE`: Retira permisos previamente otorgados.
