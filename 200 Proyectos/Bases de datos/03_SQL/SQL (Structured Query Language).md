# SQL (Structured Query Language)

**SQL** es el lenguaje de programación estándar para gestionar y consultar datos en bases de datos relacionales.

Es un **lenguaje declarativo**, lo que significa que el usuario especifica *qué* datos desea obtener, no *cómo* obtenerlos.

Se divide en varios [[Sublenguajes de SQL]] para diferentes tipos de tareas.
