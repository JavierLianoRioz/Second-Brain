# TCL (Transaction Control Language)

El **TCL** es el [[Sublenguajes de SQL|sublenguaje de SQL]] que se utiliza para gestionar las [[Transacción|transacciones]] en la base de datos.

Sus comandos aseguran la integridad de los datos cuando se ejecutan múltiples operaciones [[DML (Data Manipulation Language)]] en un solo bloque.

Ver [[Comandos de Control de Transacciones (TCL)]] para la lista de comandos.
