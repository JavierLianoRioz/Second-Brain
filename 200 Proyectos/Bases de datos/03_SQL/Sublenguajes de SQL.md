# Sublenguajes de SQL

El lenguaje [[SQL (Structured Query Language)]] se divide en varios subconjuntos de comandos según su función:

*   **[[DDL (Data Definition Language)]]:** Para definir y gestionar la estructura de la base de datos.
*   **[[DML (Data Manipulation Language)]]:** Para consultar y manipular los datos almacenados.
*   **[[DCL (Data Control Language)]]:** Para gestionar permisos y seguridad de los usuarios.
*   **[[TCL (Transaction Control Language)]]:** Para gestionar [[Transacción|transacciones]].
