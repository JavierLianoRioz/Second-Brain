# SQL DDL (Data Definition Language)

> [!WARNING]
> **NO ENTRA EN EL EXAMEN FINAL (DDL)**



El **DDL** permite definir y modificar la estructura de la base de datos (tablas, índices, restricciones). Es la implementación física del [Modelo Relacional Conceptos](../02_Dise%C3%B1o/Modelo_Relacional_Conceptos.md).

## Comandos Principales
*   [CREATE TABLE](CREATE_TABLE.md): Crea una nueva tabla.
*   `ALTER TABLE`: Modifica una tabla existente.
*   `DROP TABLE`: Elimina una tabla.
