# SQL DDL (Data Definition Language)


El **DDL** permite definir y modificar la estructura de la base de datos (tablas, índices, restricciones). Es la implementación física del [Modelo_Relacional_Conceptos](../02_Dise%C3%B1o/Modelo_Relacional_Conceptos.md).

## Comandos Principales
*   [CREATE_TABLE](CREATE_TABLE.md): Crea una nueva tabla.
*   `ALTER TABLE`: Modifica una tabla existente.
*   `DROP TABLE`: Elimina una tabla.
