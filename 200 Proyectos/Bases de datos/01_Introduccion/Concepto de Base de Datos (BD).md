# Concepto de Base de Datos (BD)

Una **Base de Datos** es una colección de **datos relacionados**, organizados y estructurados para facilitar su acceso, gestión y actualización de manera eficiente y segura.

Es gestionada por un [[Sistemas de Gestión de Bases de Datos (SGBD)]].
