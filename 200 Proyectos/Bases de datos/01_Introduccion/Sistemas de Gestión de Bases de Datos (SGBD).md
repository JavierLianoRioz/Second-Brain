# Sistemas de Gestión de Bases de Datos (SGBD)

Un **SGBD** (o DBMS en inglés) es el **software** que actúa como intermediario entre el usuario y la [[Concepto de Base de Datos (BD)|base de datos]] física.

Permite al usuario:
*   Crear y definir la estructura de la BD (usando [[DDL (Data Definition Language)]]).
*   Manipular los datos (usando [[DML (Data Manipulation Language)]]).
*   Controlar el acceso y la seguridad (usando [[DCL (Data Control Language)]]).

Ejemplos de [[Tipos de Bases de Datos|SGBD relacionales]]: MySQL, PostgreSQL, Oracle, SQL Server.
