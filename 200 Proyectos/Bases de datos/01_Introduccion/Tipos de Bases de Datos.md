# Tipos de Bases de Datos

Existen diferentes tipos de bases de datos, clasificados principalmente por su modelo de datos.

*   **Relacionales:** Basadas en el [[Modelo Relacional]]. Organizan los datos en tablas. Ej: MySQL, PostgreSQL.
*   **Jerárquicas:** Estructura de árbol. Los datos tienen una relación de padre-hijo.
*   **De Red:** Similares a las jerárquicas, pero un "hijo" puede tener múltiples "padres".
*   **Orientadas a Objetos:** Los datos se representan como objetos, con atributos y métodos.
*   **NoSQL:** No relacionales, diseñadas para gran flexibilidad y volumen. Incluyen:
    *   Documentales
    *   Clave-Valor
    *   Grafos
