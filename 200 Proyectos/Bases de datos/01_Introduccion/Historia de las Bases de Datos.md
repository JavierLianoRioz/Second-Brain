# Historia de las Bases de Datos

La evolución de las bases de datos muestra el paso desde sistemas simples a los complejos sistemas actuales.

*   **Años 60-70:** Se usaban **archivos planos**, luego surgieron los modelos [[Tipos de Bases de Datos|jerárquico (IMS)]] y de [[Tipos de Bases de Datos|red (CODASYL)]].
*   **Años 70-80:** Nace el [[Modelo Relacional]], propuesto por E.F. Codd, que se convierte en el estándar.
*   **Años 90-2000:** Aparecen las bases de datos distribuidas y la arquitectura cliente-servidor.
*   **Actualidad:** Dominan las bases de datos [[Tipos de Bases de Datos|NoSQL]], cloud y el Big Data.
