# Modelo Entidad-Relación (MER)

El **MER** es una técnica de modelado gráfico usada en el [[Pasos del Diseño de una BD|diseño conceptual]] para describir la estructura de una base de datos.

### Componentes Principales

*   **Entidad:** Objeto del mundo real sobre el que se guarda información (e.g., `CLIENTE`). Se representa con un rectángulo.
*   **Atributo:** Característica de una entidad (e.g., `nombre`). Se representa con un óvalo.
*   **Relación:** Asociación entre entidades (e.g., `CLIENTE` *compra* `PRODUCTO`). Se representa con un rombo.

El **MER Extendido** añade conceptos como entidades débiles, herencia (ISA) y cardinalidad más detallada.
