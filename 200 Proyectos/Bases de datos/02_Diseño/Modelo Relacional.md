# Modelo Relacional

Propuesto por E.F. Codd, es el modelo de datos más utilizado. Representa los datos en **tablas bidimensionales** llamadas **relaciones**.

### Componentes Clave

*   **Relación (Tabla):** Conjunto de datos sobre un tema.
*   **Tupla (Fila):** Un registro o instancia única en una tabla.
*   **Atributo (Columna):** Una propiedad o campo.
*   **[[Clave Primaria (PK)]]:** Atributo que identifica unívocamente cada fila.
*   **[[Clave Foránea (FK)]]:** Atributo que referencia la [[Clave Primaria (PK)]] de otra tabla para establecer una relación.

Este modelo es la base para los SGBD relacionales y el lenguaje [[SQL (Structured Query Language)]]. Su consistencia se mantiene mediante la [[Integridad Referencial]].
