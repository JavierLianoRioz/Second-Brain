# Atributo

Un **Atributo** es una característica o propiedad de una [[Entidad]].

## Ejemplos
*   Para `Cliente`: `nombre`, `email`, `telefono`.
*   Para `Producto`: `precio`, `stock`.

## Identificador
Un atributo especial que distingue a cada instancia de la entidad de manera única. En el modelo relacional se convierte en la [[Clave_Primaria]].
