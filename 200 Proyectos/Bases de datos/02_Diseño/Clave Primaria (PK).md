# Clave Primaria (PK)

La **Clave Primaria** (Primary Key) es un atributo o conjunto de atributos que **identifica de forma única** cada fila (tupla) en una tabla ([[Modelo Relacional]]).

### Características

*   **No puede ser nula:** Debe tener siempre un valor.
*   **Debe ser única:** No pueden existir dos filas con el mismo valor de clave primaria.

Es fundamental para establecer relaciones entre tablas a través de [[Clave Foránea (FK)|Claves Foráneas]].
