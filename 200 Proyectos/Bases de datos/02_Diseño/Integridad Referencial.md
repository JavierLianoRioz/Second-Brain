# Integridad Referencial

Es una propiedad de las bases de datos relacionales que asegura que las relaciones entre tablas se mantengan consistentes.

Gracias a las [[Clave Foránea (FK)|Claves Foráneas]], la integridad referencial garantiza que un valor de clave foránea siempre apunte a una fila existente en la tabla referenciada o sea nulo.

Esto previene registros "huérfanos" (registros que apuntan a algo que ya no existe).
